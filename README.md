<h1 align="center">_printf</h1>

<p align="center">This is a simple printf function created from scratch</p>

## Links

- [Repository](https://github.com/jtorres122/printf "<_printf> Repo")

## Available Commands

* Function:
	* _ printf();

* Format Modifiers
	* %s = string
	* %c = characters
	* %i = integer
	*  %d = integer

* Example:
	- _ printf("Hello %c\n", 'J');
